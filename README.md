# 암기빵

가장 효율적으로 외울 수 있도록 돕는 학습 앱 🍞 암기빵

## 앱 소개

### 기억률 수치
기억률 수치로 학습 성과를 쉽고 객관적으로 평가할 수 있어요.

### 다양한 문제 타입
다양한 문제 타입을 통해 어떤 종류의 외울 것이든 효율적으로 외울 수 있어요.

### 문제 세부 설정
문제 세부 설정을 통해 내 입맛대로 학습할 수 있어요.

### 커뮤니케이션
다른 사용자와 암기빵을 공유하고 평가할 수 있어요.
